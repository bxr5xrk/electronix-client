# e-commerce

Tools:
- React.js (Typescript)
- Redux, RTK Query
- Tailwind
- Vite
- Prettier, EsLint

Functionality:
- Filters (by price, category and brand)
- Full-text search
- Pagination
- Adding / removing item to WatchList / Cart (stored in localStorage)

Plans: 
- ~Responsive design~
- ~Saving filters in url params (and parsing)~
